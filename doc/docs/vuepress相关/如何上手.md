## 安装
在空目录下执行
```shell script
npm init -y
```
初始化我们的项目，然后本地安装vuepress
```shell script
npm install -D vuepress
```
## 添加必要的文件
在项目目录下添加一下目录和文件：
| 文件的相对路径        | 页面路由地址           |
| ------------------- |:---------------------:|
| /README.md          | \                     |
| /guide/README.md    | /guide/               |
| /config.md          | /config.html          |
